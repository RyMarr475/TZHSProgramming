what is git(?);

Git is Distributed Version Control System (DVCS) that makes collaborative coding easier. Well, what is a DVCS? A version control system is a system designed to allow you to keep track of changes you make to a file over time.

An example that you may have seen is an online video game that needs to patch itself. When you upgrade from patch 2.1.1 to 2.1.2, you access a central repository (a server at the game company) to get the new patch.

These systems are useful for collaborative coding because they automatically keep your project up to date. Before you begin working, you can download the latest changes that people have made. When you are done, you push the changes to the main repository so that the newest version contains your changes. The 'distributed' part just means that every computer has the entire version database rather than just the most up to date version, as in the video game example.
